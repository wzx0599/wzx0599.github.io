层次分析法

(Analytic Hierarchy Process，AHP)，主要应用在不确定情况下及具有多数个评估准则的决策问题上。

层次分析法首先需要确定相关的评价准则，这个准则的获取方式可以是查询已有的论文、根据常识和生活经验或者是咨询专家。在获取相关的评价准则后需要进行每个准则在决策时的评分权重。这需要进行一个权重确定的。这会使用到一个权重表格，然后根据重要程度对每个准则进行两两比较评分。这样就会形成一个判断矩阵，这个判断矩阵也叫正互反矩阵。但是这里有个问题，就是这个判断矩阵会出现不一致的现象，例如，判定\(A=B\)，\(B>C\)，但是在进行\(A\)和\(C\)的判断时，\(A<B\)。这显然是不符合实际情况的。这就需要我们进行判断矩阵的一致性检验。判断矩阵有一个规律，各行或者各列之间是成倍数关系的，这可以帮助我们最后进行判断矩阵的权重调整。在进行判断矩阵一致性检验时，需要进行CI，RI和CR三个变量的计算，最终要求\(CR<0.1\)，认为判断矩阵的一致性程度是可以接受的。在通过一致性检验后，需要根据判断矩阵计算各项的权重。注意，权重最好是归一化的权重。这里计算权重的方法有三种：算术均值法，几何均值法和特征值法。

同理的，每个方案的单个指标评分也是可以通过上面的过程确定的。在确定好评价表格中所有的项目后，可以计各个方案最终的得分，然后选择评分高的作为最终的优选方案。

层次分析法也可以是多层的，但是在进行判断矩阵计算时，针对的对象应该是本层对上一层的决策。同时，每一次判断矩阵的一致性检验应该是单独进行的，不可以跨层进行。如果在本层中某一项因素对上一层某个决策不产生影响，那么可以认为该因素对上层影响的权重为0，这样就可以构建相同的判断矩阵，然后进行相关的计算。

关键词

权重表格，决策类问题，评价准则，分治思想，判断矩阵，正互反矩阵，一致性检验，权重归一化，算术均值，几何均值，特征值，目标层，准则层，方案层，稳健性

TOPSIS(优劣解距离法)

(Technique for Order Preference by Similarity to an Ideal Solution，TOPSIS)，是一种常用的组内综合评价方法，能充分利用原始数据的信息，精确反映各个方案之间的差距。基本过程为基于归一化后的原始数据矩阵，采用余弦法找出有限方案中的最优方案和最劣方案，然后分别计算各评价对象与最优方案和最劣方案间的距离，获得各评价对象与最优方案的相对接近程度，以此作为评价优劣的依据。该方法对数据分布及样本含量没有严格限制，数据计算简单易行。

考虑没有待评价方案权重的情况。因为数据上有一些是极大型指标(越大越好)，有些是极小型指标(越小越好)，有些是中间型指标(越靠近一个值越好)，有些是区间型指标(落在某一个区间内最好)。为了保证处理方法的一致性，我们需要将所有类型的指标均转换为极大型指标，也一步称为正向化。之后，根据标准化操作的公式 \[ z_{ ij } = x_{ ij } / \sqrt{ \sum \limits_{ i=1 } ^ { n } { x_{ ij } ^ 2 } } \] 求解标准化后的公式，然后根据计算标准化矩阵中每个列向量元素与向量中的最大值和最小值的距离，然后进行归一化操作，得到最终的评价分数。

关键词

极大型指标，极小型指标，正向化，标准化，计算得分公式，雷达图
