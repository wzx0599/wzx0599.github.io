C++实现的AVL树

发表于 2021-05-21 更新于 2021-07-20 分类于 C++

简介

AVL是一种高度平衡的二叉搜索树，其高度平衡性主要由平衡因子和节点旋转两个部分维护。

平衡因子

平衡因子衡量的是二叉树的高度关系。在对树进行插入或删除操作时，会对子树之间的高度差产生影响。对于树中的任意一个节点，其左子树和右子树的高度差为平衡因子\(BF\)。其中，\(left\)和\(right\)分别为左子树和右子树，\(height(\cdot)\)返回的是树的高度。

\[ BF = height(left)-height(right) \]

对于AVL树而言，其平衡因子的取值仅有\(-1\)，\(0\)和\(1\)三种。

节点旋转

一个AVL树在进行节点的添加或删除操作后可能会导致平衡因子的绝对值不再小于2。此时，我们就需要通过节点的旋转操作重新维护树的平衡性。

右旋(LL)

当插入的节点位于根节点的左孩子的左子树(LL)上，需要对树进行右旋操作：将根节点放在左孩子的右子树上，将原来左孩子的右子树挂在根节点的左孩子上。

左旋(RR)

当插入的节点位于根节点的右孩子的右子树(RR)上，需要对树进行左旋操作：将根节点放在右孩子的左子树上，将原来右孩子的左子树挂在根节点的右孩子上。

先左旋再右旋(LR)

当插入的节点位于根节点的左孩子的右子树(LR)上，需要对树先左旋再右旋操作：先对根节点的左孩子进行左旋操作，然后对根节点进行右旋操作。
